To run the project in your local server:
1. Install Git Bash if not installed already
   - Windows: Navigate to https://git-scm.com/downloads/win
   - MacOS: Open the Terminal app, run 'git --version', and follow further instructions
   - Linux: Open the Terminal app and run 'sudo apt update' & 'sudo apt install git'
2. Go to python.org to download the latest version of Python
2. Open the Terminal app in the desired directory (i.e. downloads directory would be accessed by running 'cd ~/Downloads')
3. Run 'git clone https://github.com/anaym952/AV-Math-Team-Website.git'
4. Open a new terminal in the project folder in the directory in which it was cloned
